# GraphOfSets

Olá
Para executar esse código, precisa ter instalado em sua máquina a linguagem Python.
(Usando o VSCode)
1 - Abra um novo terminal em: Terminal -> Novo Terminal;
2 - Digite python -m http.server para abrir um servidor local;
3 - O servidor usará a porta 8000, logo você pode abrir qualquer aquivo dessa pasta usando "http://localhost:8000/JS/Graph/nome_do_arquivo.extensão";
4 - Para ver o funcionamento do código grafo.js com html, digite http://localhost:8000/JS/Graph/index-graph.html;
5 - Para finalizar a visualização, no terminal python, use o comando Ctrl+C para finalizar e feche a página web.

Caso haja algum erro na visualização da página web, verifique se o endereço está correto
